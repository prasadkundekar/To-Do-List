📝 To-Do List Web App

A responsive **To-Do List** application built with HTML, CSS, and Vanilla JavaScript. Manage tasks with due dates, priorities, dark/light mode, and persistent storage — all in a clean, minimal interface.

✨ Features
- ✅ Add, edit, delete, and mark tasks as complete  
- 📅 Due date selection with Flatpickr  
- 🎯 Task priorities with color indicators  
- 🌗 Dark/Light mode toggle (saved in `localStorage`)  
- 🔍 Search and sort tasks for quick organization  
- 💾 Persistent storage using `localStorage`  
- ⬆ Scroll-to-top button with smooth animation  
 

📂 Project Structure
index.html # UI layout & structure
style.css # Styling & themes
index.js # Main application logic
scroll.js # Scroll-to-top button functionality
